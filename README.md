# GFF2-4A

## スライムアクション
このゲームは様々な要素やエネミーを駆け巡り、ステージをクリアしたタイムを競い合うゲームです。
<br><br>

## 動作環境
 **dxlib**が`C:\PG\dxlib`にある事。  
 コントローラーの入力方式：XInputのみ対応。
 
## 操作方法
### タイトル画面
|入力 |操作		|
|:-----:|:-------------------------:|
|アナログスティック上下|選択	|
|B   |決定	|
|BACK|ゲーム終了	|

<br>

### ゲーム画面
|入力 |操作		|
|:-----:|:-------------------------:|
|アナログスティック左右|プレイヤー左右移動	|
|START|ポーズ	|
|BACK|ゲーム終了	|

<br>
